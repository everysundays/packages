Rack & Rail Grid System

A responsive grid system with two container types: Rack (end-to-end columns that wrap) and Rail (horizontally scrollable rows).

Breakpoint Reference

sm: 375px viewport (335px available)

md: 770px viewport (690px available)

lg: 1450px viewport (1386px available)

xl: 1800px viewport (1736px available)
