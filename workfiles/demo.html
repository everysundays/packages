Customization Guide

The grid system constants can be easily modified to adjust col-ratios and other properties.

How to Customize:
  • Edit the src/scripts/grid-config.js file to adjust col-percentages, widths, and spacing
  • Modify RACK_col- to change the percentage-based col-widths
  • Modify RAIL_col- to change the fixed-width col- in the rail container
  • Adjust SYSTEM.CONTAINER_PADDING to change the standard padding (currently 1.5rem)
  • Adjust SYSTEM.GAP to change the standard gap between col- (currently 1rem)
  • The GRID_COLORS object allows customization of the visual appearance
Project Structure:
  • src/scripts/grid-config.js: Contains all configuration settings
  • src/styles/main.css: Core grid system CSS
  • src/styles/demo.css: Demo styling (not part of the core system)
  • src/scripts/demo.js: Demo page functionality
