Customization Guide

The grid system constants can be easily modified to adjust column ratios and other properties.

How to Customize:
  • Edit the src/scripts/grid-config.js file to adjust column percentages, widths, and spacing
  • Modify RACK_COLUMNS to change the percentage-based column widths
  • Modify RAIL_COLUMNS to change the fixed-width columns in the rail container
  • Adjust SYSTEM.CONTAINER_PADDING to change the standard padding (currently 1.5rem)
  • Adjust SYSTEM.GAP to change the standard gap between columns (currently 1rem)
  • The GRID_COLORS object allows customization of the visual appearance
Project Structure:
  • src/scripts/grid-config.js: Contains all configuration settings
  • src/styles/main.css: Core grid system CSS
  • src/styles/decoration.css: Demo styling (not part of the core system)
  • src/scripts/demo.js: Demo page functionality
