# PSP Designer Tools A comprehensive toolkit for frontend developers with a responsive grid system and development environment setup. ## 📦 Packages This monorepo contains two main packages: ### 🎨 [@psp/layoutsystem](./packages/layoutsystem/) Responsive grid system for Tailwind CSS with **Rack** and **Rail** containers - Flexible wrapping columns (Rack) - Horizontal scrolling rows (Rail) - Extended breakpoints for ultra-wide screens - Mathematical grid with precise offsets ### 🏗️ [@psp/environment](./packages/environment/) Node.js development environment setup for frontend projects - Project templates and scaffolding - Development scripts and tooling - Build configurations ## 🚀 Quick Start ### Install Specific Package ```bash # Layout system only npm install @psp/layoutsystem # Development environment only npm install @psp/environment # Both packages npm install @psp/layoutsystem @psp/environment ``` ### Development Setup ```bash # Clone the entire repository git clone https://github.com/everysundays/psp-designertools.git cd psp-designertools # Install all dependencies npm install # Build all packages npm run build # Start development mode npm run dev ``` ### Get Only What You Need ```bash # Get only the layout system git clone --filter=blob:none --sparse https://github.com/everysundays/psp-designertools.git cd psp-designertools git sparse-checkout init --cone git sparse-checkout set packages/layoutsystem # Get only the environment tools git sparse-checkout set packages/environment ``` ## 📋 Available Scripts ```bash npm run build # Build all packages npm run dev # Start development mode for all packages npm run clean # Clean all build artifacts npm run lint # Lint all packages npm run test # Run tests for all packages npm run demo # Open layout system demo ``` ## 🎨 Layout System Usage ### Rack Container (Flexible Wrapping) ```html
