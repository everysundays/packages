The CUBE

Compact appliance available on a rental basis that enables medical institutions of any size to assemble PACS of desired scale. SSD storage ensures high fault-tolerance.

Smart-retrieve

Ensures high speed image access while minimizing disk usage. Prefetches data from centers in advance based on EHR and ordering system data.

Collection & Sharing

Worklist Integration

DICOM Viewer

Standard & Standalone Viewer

The main DICOM Viewer, fully equipped with diagnosis functions for radiologists.

  • Multimodality - One viewer for DICOM and non-DICOM
  • Customizability - Configure modality-specific protocols
  • Interpretation tools - Apply image reconstruction
  • Export and sharing - Burn DICOM data to CDs/DVDs

Web-based Viewer

Zero-footprint web viewer for instant access to medical images from any device with browser support.

Administrator Tools

Importer

Tool to import DICOM images (from CD/DVD, hard disks) or non-DICOM data (images such as PNG or JPG, document like PDF) into PACS.

Mobile Viewer

Mobile-optimized viewer for on-the-go access to medical images with touch-friendly controls and offline capabilities.

DICOM Tag Editor

Efficient tool to edit DICOM tag information, merge/split series, delete studies before or after sending to PACS. The system offer more options during the Quality Assurance process in the radiology workflow

  • Can receive images directly from modality so that radiographers can confirm, edit the DICOM files before sending to PACS.
